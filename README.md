# Task 3: Networking Basics for Cyber Security

## Objective
To understand basic networking concepts and analyze live network traffic using Wireshark.

## Tools Used
- Kali Linux
- Wireshark

## Task Performed
- Installed and configured Wireshark
- Captured live network traffic
- Applied filters for HTTP, DNS, TCP
- Observed TCP three-way handshake
- Identified encrypted (HTTPS) and plain-text (HTTP) traffic
- Saved packet capture for analysis

## Files Included
- network_capture.pcapng – Packet capture file
- analysis_report.md – Traffic analysis and observations
- screenshots/ – Evidence of Wireshark usage

## Learning Outcome
Gained hands-on experience in packet sniffing and network traffic analysis.
